记录vps的购买到ss的搭建

发表于 2019-03-15 更新于 2020-08-12 分类于 VPS

每个程序员都希望自己有一台私人的主机，也或许国内的屁民想看看YouTube等放松下自己、或者查看国外其他技术文档时出现404、”Sorry, XXX is not available in your country yet.”等这样就很烦了！购买一台国外的VPS服务器就可以完美解决上面两个问题。

VPS服务器

VPS服务器需要选择国外的，首选国际知名的vultr，装上bbr速度不错、按小时计费，删除即停止计费，也不用担心IP被封，就是有点小贵现在最便宜的5刀一月。推荐他人使用会有10刀，好吧这是推介地址vultr。自己用的是电信的宽带，vps地址选择的是日本ping在150左右，试了下联通的ping在80左右，对于来玩游戏说是肯定不行的。网上的vps购买教程有很多，且支持支付宝，就不再累述了。

BBR加速

如果到此时你访问YouTube的网站会发现还是很卡，一般只有3/4百k的速率。
Google 开源了其 TCP BBR 拥塞控制算法，并提交到了 Linux 内核，从 4.9 开始，Linux 内核已经用上了该算法。所以要使用此功能需要查看自己系统的内核是否在之上，内核的升级办法可以参考[2]。

  1. uname -r 查看内核

  2. wget --no-check-certificate https://github.com/teddysun/across/ raw/master/bbr.sh && chmod +x bbr.sh && ./bbr.sh 以root用户执行安装脚本、需要重启

  3. sysctl net.ipv4.tcp_available_congestion_control 查看是否有bbr字样，有表示安装成功

  4. lsmod | grep bbr 返回值有 tcp_bbr 模块即说明 bbr 已启动
    现在来看看看视频的速率有3M/s了，真的是数量级上的提升。也可以选择使用 kcptun 来加速，相比较bbr有点麻烦。

总结

  • vultr知名厂商
  • 按小时计费，删除服务器及停止
  • 装上bbr或者kcptun在YouTube上看1080P视频没压力
  • 有点小贵
